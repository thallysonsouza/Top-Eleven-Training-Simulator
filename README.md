# ⚽ Top Eleven Training Simulator

Simulador desenvolvido em Excel para gerar automaticamente sequências de treino no Top Eleven.

## Funcionalidades

- Inserção das habilidades iniciais
- Seleção da posição do jogador
- Definição do GRL desejado
- Definição da média de treino
- Geração automática da sequência de treinos
- Exibição das habilidades finais
- Suporte para duas sequências (A e B)
