# ⚽ Top Eleven Training Simulator

An advanced Microsoft Excel simulator that automatically generates optimized training sequences for **Top Eleven** players based on their initial attributes, position, target overall rating (GRL), and desired training average.

---

## 📋 Overview

Training players efficiently in Top Eleven can be time-consuming, especially when trying to maximize key attributes while maintaining an optimal overall rating.

This simulator automates the entire process by calculating and recommending the most suitable training sequence according to the player's profile.

The spreadsheet was developed using advanced Excel formulas, validation rules and optimization logic to provide a practical and intuitive experience.

---

# ✨ Features

* ✅ Automatic training generation
* ✅ Support for multiple player positions
* ✅ Initial player attribute input
* ✅ Target GRL configuration
* ✅ Desired training average configuration
* ✅ Two alternative training sequences (Option A and Option B)
* ✅ Complete simulation of final player attributes
* ✅ Exercise-by-exercise visualization
* ✅ Automatic calculation of white skills average
* ✅ Input validation to prevent invalid values
